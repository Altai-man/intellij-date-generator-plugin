Current Date Generator Plugin
=====

A simple plugin for Intellij IDE's which generates the current date in different formats.

### Features

* Multi caret support

### Supported formats

* ATOM (2003-12-13T18:30:02+01:00)

### Usage

<kbd>cmd</kbd> + <kbd>N</kbd> shows the `Generate date` action.
